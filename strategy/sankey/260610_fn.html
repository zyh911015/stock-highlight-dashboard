这张图里两条河同时汇向市值:① 利润河:收入 → 净利润 →(加"预期放大器")→ 故事溢价;② 家底河:现金 + 其他资产 → 总资产 → 分成负债和账面净资产。流进 = 流出,完全合规。每块标 $金额·占市值%。每块停鼠标看名词解释。

名词速查(怕你不熟): 净利润=扣完所有成本费用税真正赚到的钱;账面净资产=总资产−总负债,清算后剩给股东的"看得见的家底";故事溢价=市值里超出家底的部分,市场为"未来"付的钱;预期放大器=故事溢价里"利润之外、靠预期多付"的那块;市值=股价×总股本。
怎么读这张整合图:
· 利润河(上):年收入 $4.24B 扣成本和税剩净利润 $0.42B;净利润 + "预期放大器"$18.3B = 故事溢价 $18.7B → 市值。(放大器 = 市场基于 AI 光模块成长预期多付的钱。)
· 家底河(下):现金+短投 $0.95B + 其他资产 $2.56B = 总资产 $3.51B → 负债 $1.20B + 账面净资产 $2.30B;账面净资产 → 市值。
· 两河在"市值"$21.0B 汇合:账面家底只占 11%,其余 89% 是"利润 × 预期"的故事。放大器($18.3B)是净利润的 43 倍——意思是:按现在的年利润,要 43 年才能"赚回"市场预付的预期;市场赌的是利润未来几年翻几倍,把 43 年缩短到十几年。
🧬 第一性原理:为什么"收支"早就和"资产、市值"连在一起
① 资产负债表 = 历年利润表的累积:FN 上市 15 年净利润没分掉的部分 → 留存收益 $2.43B → 变成泰国的厂房、设备、存货和现金。
② 市值 = 过去利润(账面净资产 $2.3B)+ 未来利润的折现(溢价 $18.7B)。
链条:收入 − 支出 = 净利润 → 留存 → 变资产 → 撑起账面净资产 → 市值的"家底";同时 未来净利润 → 折现 → 溢价 → 市值的"预期"。FN 这一年的大涨,几乎全涨在"预期"那块——所以它对"预期落空"也最敏感(财报 beat 都能跌 8–13%,见"④ 推演")。

整个生意值多少钱(企业价值 EV)

从"股东那部分"(市值)出发,加上要替它还的债、减去账上现金 = "连债主一起算、整个生意的总价"。每根柱子都标了金额。

What the $21.0B market cap is made of

市值 = 账面净资产 + 市场溢价。下面那条横条让比例一目了然。

book value 是哪来的?

来自资产负债表:账面净资产 = 总资产 $3.51B − 总负债 $1.20B = $2.30B。它就是资产负债表里的"Shareholders' Equity"(股东权益),在这张市值全景图里直接流进市值。

什么是 Enterprise Value(企业价值,EV)?有什么用?
市值只算"股东那部分";EV 算的是"整个生意"值多少钱——把债主也算进来。公式:EV = 市值 + 有息债 − 现金。
· 为什么 +债:整体收购一家公司,除了买下股票,还得替它还债,所以债要加上。
· 为什么 −现金:账上的现金收购后就归买家了,等于变相打折,所以减掉。
FN:$21.0B 市值 + 几乎为零的债($4M 租赁)− $0.95B 现金短投 = EV ≈ $20.1B。它是少见的"净现金"公司,EV 比市值还低 ~$0.9B。
意义:比较两家公司贵不贵时,EV 比市值更公平——剔除了"谁借的债多、谁现金多"的干扰,只比"生意本身"的价钱。

这只股票该值多少钱?(公允价值 · 正当倍数法)

不用 DCF(要猜 10 年增速、太黑箱),改用更透明的正当倍数法:公允价 = 前瞻每股收益 $17.27(FY2027 non-GAAP 一致预期,9 位分析师)× 一个"该给的倍数"。倍数从"同行给多少倍 + FN 的质量分 − 集中度/代工模式扣分"推出来(熊 20× / 中 28× / 牛 38×)。红线 = 今天 $586(33.9×)。

公允价:$345(熊 20×)/ $483(中 28×)/ $656(牛 38×)。今天 $586 落在"中性偏牛"之间——市场已经按"供应解除、1.6T 顺利接棒、大客户不变心"的剧本在定价。
倍数怎么定的:纯 EMS 代工同行(Jabil 25×、Celestica 25×、Flex 21×、Sanmina 19×)中枢约 19–25×;光器件设计商(Coherent 44×、Lumentum 45×)有自有芯片/IP,贵一档。熊档 20× = "它终究是代工厂"(增速回落到 10–15% 时市场会这么给);中档 28× = EMS 上限 + 独家光封装工艺 + 25% 增速的溢价;牛档 38× = 向设计商倍数靠拢(需要增速持续 25%+、供应约束解除、客户继续外包)。
这个方法把真正的赌点逼出来了:① $17.27 的 FY27 预期 EPS 靠不靠谱(供应约束 + 客户集中是两大变数);② FN 配不配"超越代工"的倍数。对照:分析师目标价低 $635 / 中位 $750 / 高 $850——华尔街比本模型的中性档乐观得多(中位 $750 ≈ 43× 前瞻,直接按设计商倍数给)。今天的价 = 市场押在"分析师对、模型保守"那一边。
